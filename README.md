# online-store-dm102
Java Online Store

This is a Java application that simulate users to browse and purchase items from an online store. The application is built using Java 19.

<img alt="Online Store UML" src="/UML/OnlineStore_UML.png">

## Running the application

To run the application, you need to have Java 19 installed on your machine. You can download it from [here](https://www.oracle.com/java/technologies/javase-downloads.html).

Also, you need to have Lombok installed in your IDE. You can download it from [here](https://projectlombok.org/download).

Configure your IDE to use Lombok. You can follow the instructions from [here](https://projectlombok.org/setup).

Once you have Java and Lombok installed, you can run the application by running the *src\com\inatel\dm102\onlinestore\OnlineStoreApplication.java*.
